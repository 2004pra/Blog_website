## Key Technical Decisions

### Backend Architecture (Human-Written)
The backend of this project was **entirely developed manually** by me. No AI tools were used in the creation of the backend logic, authentication, database setup, or API endpoints. 

To ensure a lightweight and high-performance application, the backend was efficiently built using **Flask**. For handling user authentication securely, **JWT (JSON Web Tokens)** and **bcrypt** were implemented for session management and password hashing, alongside other essential Python libraries.

### Frontend Architecture (AI-Generated)
The frontend of this project was generated and built with the assistance of AI, specifically using **Claude Haiku 4.5**. The AI was utilized to create the standard React components, CSS styling, state management (AuthContext), and to establish seamless connections to my Flask backend API. 

## Project Structure
- `Backend/`: Python backend logic and API. (Do not use AI for this folder)
- `Frontend/`: React frontend application. (Generated via Claude Haiku 4.5)

For more details on how AI was used and the rules for future AI generation, please refer to `AI_GUIDANCE.md`.
